# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Email: contact@silverstream.tech

Include:
- Description of the vulnerability
- Steps to reproduce
- Relevant logs or screenshots

## Response Timeline

- Acknowledgement: within 48 hours
- Initial assessment: within 5 business days
- Fix or mitigation: as soon as reasonably possible

## Scope

This policy covers the NexusFIX library (`include/nexusfix/`).

## Disclosure

Coordinated disclosure. Do not open public issues for security vulnerabilities.
